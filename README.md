# DS
# Data science course on ITAcademy Barcelona.

## Presentation of the course
The main objective of this course is to learn the basic concepts of data science and achieve a
effective competence with Python code, oriented to data analytics.
The course is aimed at students with basic knowledge of statistics, SQL and python.
Work is done in the Jupyter Notebook environment. In the first part of the course, databases will be worked on
with MySQL, numerical programming tools, dataframes (similar to Excel tables) and the
data visualization packages.
In the following modules, Machine Learning will be introduced and learning algorithms will be worked on
supervised and unsupervised. Finally, tools such as pipelines, text mining tools and
web scraping techniques.
Once the different modules have been completed and the associated tasks have been presented, the students will move on to
carry out a transversal final project in which to apply the knowledge gained.


## Course contents
The course has an itinerary of 10 thematic modules and a transversal final project. Everyone
of these modules has different self-learning resources and tasks associated with each
one of the modules.
- Module 0. Presentation of the course and installation of the work environment: in this module it will be explained what
is and what a work environment is for and the Jupyter Notebook environment will be introduced and installed and
Github to deliver the exercises.
- Module 1. Relational databases (SQL): in this module you will see basic concepts of
SQL and relational databases will be created and queried using MySQL.
- Module 2. Python basics: basic Python concepts such as the structures of
data and control structures.
- Module 3. Numerical programming, dataframes and statistical analysis: in this module the
basic python packages that allow statistical treatment of data as well as work
and arrangement in tables. The NumPy and Pandas libraries are introduced.
- Module 4. Graphical visualization of data: in this module the main packages are worked on
allow the data to be visualized graphically. Specifically the Matplotlib and Seaborn packages.
- Module 5. Introduction to Machine Learning: the first machine concepts are introduced
learning and one of the main data science packages with python is presented: scikitlearn.
- Module 6. Supervised learning algorithms: Regression: this module presents i
they work on the main regression algorithms, their presentation and their evaluation.
- Module 7. Supervised learning algorithms: Classification: this module presents i
they work on the main classification algorithms, their presentation and how they are evaluated.
- Module 8. NON-supervised learning algorithms: Grouping: they are presented and worked on
main grouping algorithms, how they are presented and how they are evaluated.
- Module 9. Analysis of sentiment and texts: in this module text mining is introduced and worked on.
- Module 10. Web scraping and automation: in this module the main ones are presented and worked on
web scraping packages for Python and is put into practice with different exercises.
- Final project: the final project must have a professional aspect. This means that it has
of being focused on a real case or interest. The project must have, at least, the
following sections:
  + Presentation, contextualization and interest of the case.
  + Description of the data.
  + Statistical and graphical analysis of the most relevant data.
  + Application of algorithms seen throughout the course and which may be useful in the case.
  + Analysis of the extracted information.
  + Conclusions





# Curs de ciència de dades a ITAcademy Barcelona.

## Presentació del curs
L’objectiu principal d’aquest curs és conèixer els conceptes bàsics de ciència de dades i assolir una
competència efectiva amb codi Python, orientat a l’analítica de dades.
El curs està orientat a alumnes amb coneixements bàsics d’estadística, SQL i python.
Es treballa a l’entorn Jupyter Notebook. En la primera part del curs es treballaran les bases de dades
amb MySQL, les eines de programació numèrica, els dataframes (similars a taules d’excel) i els
paquets de visualització de dades.
En els següents mòduls s’introduirà el Machine Learning i es treballaran els algoritmes d’aprenentatge
supervisat i no supervisat. Finalment es presentaran eines com els pipelines, eines de text minning i
tècniques de web scraping.
Un cop finalitzats els diferents mòduls i presentades les tasques associades, els alumnes passaran a
realitzar un projecte final transversal on aplicar els coneixements assolits.


## Continguts del curs
El curs compta amb un itinerari de 10 mòduls temàtics i un projecte final transversal. Cada un
d’aquests mòduls compta amb diferents recursos d’autoaprenentatge i tasques associades a cada
un dels mòduls.
- Mòdul 0. Presentació del curs i instal·lació de l’entorn de treball: en aquest mòdul s’explicarà què
és i per a que serveix un entorn de treball i s’introduirà i instal·larà l’entorn Jupyter Notebook i
Github per a fer el lliurament dels exercicis.
- Mòdul 1. Bases de dades relacionals (SQL): en aquest mòdul es veuran conceptes bàsics de
SQL i es crearan i consultaran bases de dades relacionals amb MySQL.
- Mòdul 2. Bàsics de Python: es treballen conceptes bàsics de Python com les estructures de
dades i les estructures de control.
- Mòdul 3. Programació numèrica, dataframes i anàlisi estadístic: en aquest mòdul es treballen els
paquets bàsics de python que permeten un tractament estadístic de les dades així com el treball
i disposició en taules. Es presenten les llibreries NumPy i Pandas.
- Mòdul 4. Visualització gràfica de dades: en aquest mòdul es treballen els principals paquets que
permeten visualitzar les dades gràficament. En concret els paquets Matplotlib i Seaborn.
- Mòdul 5. Introducció al Machine Learning: s’introdueixen els primers conceptes de machine
learning i es presenta un dels paquets principals del data science amb python: scikitlearn.
- Mòdul 6. Algoritmes d’aprenentatge supervisat: Regressió: en aquest mòdul es presenten i
treballen els principals algoritmes de regressió, la seva presentació i la seva avaluació.
- Mòdul 7. Algoritmes d’aprenentatge supervisat: Classificació: en aquest mòdul es presenten i
treballen els principals algoritmes de classificació, la seva presentació i com s’avaluen.
- Mòdul 8. Algoritmes d’aprenentatge NO supervisat: Agrupació: es presenten i treballen els
principals algoritmes d’agrupació, com es presenten i com s’avaluen.
- Mòdul 9. Anàlisi de sentiment i textos: en aquest mòdul s’introdueix i es treballa el text minning.
- Mòdul 10. Web scraping i automatització: en aquest mòdul es presenten i treballen els principals
paquets de web scraping per Python i es posa en pràctica amb diferents exercicis.
- Projecte final: el projecte final ha de tenir una vessant professionalitzadora. Això vol dir que ha
d’estar enfocat a una casuística o interès real. El projecte ha de comptar, com a mínim, amb els
següents apartats:
  + Presentació, contextualització i interès del cas.
  + Descripció de les dades.
  + Anàlisi estadístic i gràfic de les dades més rellevants.
  + Aplicació d’algoritmes vistos al llarg del curs i que puguin resultar útils al cas.
  + Anàlisi de la informació extreta.
  + Conclusions

# Curso de ciencia de datos en ITAcademy Barcelona.

## Presentación del curso
El objetivo principal de este curso es conocer los conceptos básicos de ciencia de datos y conseguir una
competencia efectiva con código Python, orientado a la analítica de datos.
El curso está orientado a alumnos con conocimientos básicos de estadística, SQL y Python.
Se trabaja en el entorno Jupyter Notebook. En la primera parte del curso se trabajarán las bases de datos
con MySQL, las herramientas de programación numérica, los dataframes (similares a tablas de excel) y los
paquetes de visualización de datos.
En los siguientes módulos se introducirá el Machine Learning y se trabajarán los algoritmos de aprendizaje
supervisado y no supervisado. Por último se presentarán herramientas como los pipelines, herramientas de texto minning y
técnicas de web scraping.
Una vez finalizados los diferentes módulos y presentadas las tareas asociadas, los alumnos pasarán a
realizar un proyecto final transversal donde aplicar los conocimientos adquiridos.


## Contenidos del curso
El curso cuenta con un itinerario de 10 módulos temáticos y un proyecto final transversal. Cada uno
de estos módulos cuenta con diferentes recursos de autoaprendizaje y tareas asociadas a cada
uno de los módulos.
- Módulo 0. Presentación del curso e instalación del entorno de trabajo: en este módulo se explicará qué
es y para que sirve un entorno de trabajo y se introducirá e instalará el entorno Jupyter Notebook y
Github para realizar la entrega de los ejercicios.
- Módulo 1. Bases de datos relacionales (SQL): en este módulo se verán conceptos básicos de
SQL y se crearán y consultarán bases de datos relacionales con MySQL.
- Módulo 2. Básicos de Python: se trabajan conceptos básicos de Python como las estructuras de Python
datos y las estructuras de control.
- Módulo 3. Programación numérica, dataframes y análisis estadístico: en este módulo se trabajan los
paquetes básicos de python que permiten un tratamiento estadístico de los datos así como el trabajo
y disposición en tablas. Se presentan las librerías NumPy y Pandas.
- Módulo 4. Visualización gráfica de datos: en este módulo se trabajan los principales paquetes que
permiten visualizar los datos gráficamente. En concreto, los paquetes Matplotlib y Seaborn.
- Módulo 5. Introducción al Machine Learning: se introducen los primeros conceptos de machine
learning y se presenta uno de los paquetes principales del data science con python: scikitlearn.
- Módulo 6. Algoritmos de aprendizaje supervisado: Regresión: en este módulo se presentan y
trabajan los principales algoritmos de regresión, su presentación y evaluación.
- Módulo 7. Algoritmos de aprendizaje supervisado: Clasificación: en este módulo se presentan y
trabajan los principales algoritmos de clasificación, su presentación y cómo se evalúan.
- Módulo 8. Algoritmos de aprendizaje NO supervisado: Agrupación: se presentan y trabajan los
principales algoritmos de agrupación, cómo se presentan y cómo se evalúan.
- Módulo 9. Análisis de sentimiento y textos: en este módulo se introduce y se trabaja el texto minning.
- Módulo 10. Web scraping y automatización: en este módulo se presentan y trabajan los principales
paquetes de web scraping por Python y se pone en práctica con diferentes ejercicios.
- Proyecto final: el proyecto final debe tener una vertiente profesionalizadora. Esto significa que debe
de estar enfocado a una casuística o interés real. El proyecto debe contar, como mínimo, con los
siguientes apartados:
  + Presentación, contextualización e interés del caso.
  + Descripción de los datos.
  + Análisis estadístico y gráfico de los datos más relevantes.
  + Aplicación de algoritmos vistos a lo largo del curso y que puedan resultar de utilidad al caso.
  + Análisis de la información extraída.
  + Conclusiones
